Chapter 1 The world of the Lattice

As we all know, a vector can be anything in our world. So if we want to describe linear space $V$ , we need to find out two Basis Vector, for example $\vec{i}$ and $\vec{j}$, so that any vector can be shown by linear combination with them, as the same as $\vec{x}=a\cdot\vec{i}+b\cdot\vec{j}$. The set of the vector space created by $\vec{i}$ and $\vec{j}$ is this linear space $V$. But in that case, all the ‘$a$’s and ‘$b$’s can be any number, so might as well let us make $a$ and $b$ both are integers, then we got our Integer Lattice.

We can make sure that all the vector starts from the origin, and all the vector can be distingguished by the end dot, then, our linear vector space $V$ is just like many dots placed neatly and discretely. By the way, you can also do some linear transformation to the easy cartesian coordinate system and create a new lattice.

Think that now we have a lattice and its bases $\vec{b_1},\vec{b_2},\cdots,\vec{b_n}$, so all the members of this lattice can also be shown as a matrix $A$, which is
$$
A=\sum^n_{i=1}(b_i\cdot \mathbb{Z})
$$
Of course, you can also integrate all the base vectors as a matrix $B$, which means
$$
A=\sum^n_{i=1}(b_i\cdot\mathbb{Z})=B\cdot x(x\in\mathbb{Z}^n)
$$
So what is lattice? We can say lattice is just a set of vectors, but vector can be interesting.

Chapher 2 Notions in Lattice

Determinant

Determinant can be seen as a very special proportion — the proportion by a change of a linear transformation in the area of a quadrilateral or the volume of a polyhedron constituted by all the base vector.

Shortest distance

Imagine there is a lattice based on the matrix $B$, and we often use $\lambda$ to describe the shortest distance in a lattice. It’s easy to see that
$$
\lambda_1=min||x-y||\ \ \ (x,y\in B,x\not=y)\\
=min||x||\ \ \ (x\in B,x\not=\vec{0})~~~~~~
$$
What’s more, we choose $\lambda_2,\lambda_3,\cdots,\lambda_n$ to describe the $i$st shortest distance in $B$, so there’s no doubt that
$$
\lambda_1\leq\lambda_2\leq\lambda_3\leq\cdots\leq\lambda_n
$$

Chapher 3 Problems in Lattice

SVP (Shortest Vector Problem)

Let’s talk about a lattice $L$ based on matrix $B$, so try to find out a vector which fits

$$
||B\vec{x}||= \lambda_1\ \ \ (x\in \mathbb{Z}^k)
$$
It doesn’t sounds like a tough deal, isn’t it? But when we get a bad pair of vectors as bases, $SVP$ is a big problem. However, ways usually can be found in easier problem, $SVP$ does it so, which is called $SVP_\gamma$. In $SVP_\gamma$ problem, we only need to find a vector
$$
||B\vec{x}||\leq \gamma\lambda_1\ \ \ (x\in \mathbb{Z}^k)
$$

CVP (Closest Vector Problem)

This problem is in a lattice $L$ based on matrix $B$, and try to find a vector closest to this point $t$, which means
$$
||B\vec{x}-\vec{t}||\leq\mu
$$

$CVP$ also have an easier problem called $SVP_\gamma$, which means
$$
||B\vec{x}-\vec{t}||\leq \gamma\mu\ \ \ (x\in \mathbb{Z}^k)
$$

Chapher 4 Geometric structure in Lattice

Try to solve nphard problems

So as we know that $CVP$ is a hard problem, but sometimes isn’t. Imagine a Littice based in cartesian coordinate system. which means all the base vectors are vertical to each other(orthogonal basis). We find that it is quite easy to solve $SVP$ or $CVP$(round up and down), which means if we can change a lattice $\Lambda$ into a lattice $L$ with almost orthogonal basis, it will be easier for us to solve the nphard problem. By the way, we named the process ‘Lattice Basis Reduction’, such as Gram-Schmidt. But the answer we get by round up and down in Gram-Schmidt isn’t truly right, Only $||\vec{t}-\vec{v}||\leq min\frac{||\vec{b_i}||}{2}$ we can get the right answer().
